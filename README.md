# Tic-Tac-Toe Advanced Edition 🎮

A modern and feature-rich Tic-Tac-Toe game built using pure HTML, CSS, and JavaScript. This project transforms the classic game into an interactive experience with advanced gameplay options, AI opponents, customizable themes, sound effects, animations, and match history tracking.

## Features

✨ **Multiple Game Modes**

* Player vs Player (PvP)
* AI Easy Mode
* AI Medium Mode
* AI Hard Mode (Minimax Algorithm)
* AI vs AI Simulation

🎨 **Customizable Experience**

* Multiple board themes (Dark Neon, Midnight Blue, Forest, Crimson, Light)
* Custom player names
* Custom player symbols
* Adjustable AI speed

🧠 **Smart AI**

* Easy mode with random moves
* Medium mode with mixed strategy
* Hard mode powered by the Minimax algorithm with Alpha-Beta pruning for optimal decision-making

📊 **Game Statistics**

* Live scoreboard
* Win, loss, and draw tracking
* Match history log with timestamps

🔊 **Enhanced User Experience**

* Dynamic sound effects generated using the Web Audio API
* Smooth animations and visual feedback
* Turn indicators and active player highlighting
* Winning combination animations
* Last move highlighting option

⚙️ **Additional Controls**

* Undo move functionality
* New game restart
* Complete game reset
* Settings panel for customization

## Technologies Used

* HTML5
* CSS3 (Custom Properties, Animations, Responsive Design)
* Vanilla JavaScript (ES6+)
* Web Audio API

## Highlights

This project demonstrates advanced front-end development concepts, including responsive UI design, game state management, AI implementation using Minimax, DOM manipulation, theme switching, and interactive user experience design—all without relying on external JavaScript frameworks.

A perfect project for learning game development fundamentals, JavaScript algorithms, and modern UI/UX practices.
